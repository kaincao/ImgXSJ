当接手一个新项目，面对大量没有注释说明的代码，要去理解当中业务处理逻辑，既耗时又耗力。 最近在 GitHub 上发现了 Davia 这个开源工具，可用它为我们的代码库生成直观可视化的文档。 只需指向项目路径并描述要生成什么文档，它就会自动分析代码并实时生成文档页面。 GitHub：[http://github.com/davialabs/davia](https://t.co/iZO50b2zz6) 除此之外，还支持类似 Notion 的编辑体验，以及可交互的白板图表，瞬间让让代码结构和逻辑一目了然。 通过克隆代码到本地，安装相关依赖后，配置 API 密钥即可使用，支持 Anthropic、Google 等主流 AI 模型服务商。
